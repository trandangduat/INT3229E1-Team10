## Quy tắc phân nhánh (Branching)

Tuyệt đối không ai code thẳng lên nhánh main.

- **main**: Nhánh chứa code hoàn chỉnh, chạy lỗi là rớt môn.
- **dev**: Nhánh gộp code từ các tính năng, dùng để test chung.
- **feature/\<tên-tính-năng\>**: Ví dụ `feature/setup-pyspark`, `feature/ingest-mimic`.
